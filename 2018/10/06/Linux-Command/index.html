scp 远程用户名@IP地址:文件名1 本地地址文件名2
可能有用的几个参数 :
-v 和大多数linux命令中的-v意思一样,用来显示进度.可以用来查看连接/认证/或是配置错误.
-C 使能压缩选项.
-P 选择端口.注意-p已经被rcp使用.
-4 强行使用IPV4地址.
-6 强行使用IPV6地址.
